## Common Questions

### Q: Do I need to open-source my project if I use this framework?

**A: NO!** The framework is a **tool**, not a library. Your code stays private.

**Think of it like Git:**
- Git is GPL v2
- Do companies open-source their code because they use Git? NO!
- Same applies here - use the framework freely, keep your code private

### Q: When does GPL v3 require me to share code?

**A: Only if you:**
1. **Modify** the framework itself (hooks, scripts, rules)
2. **AND distribute** your modified version (sell, publish, give to others)

**If you use as-is:** No obligations whatsoever.

**If you modify privately:** No obligations (private use is free).

### Q: Can I use for commercial projects?

**A: YES!** Commercial use is fully allowed. Use in:

- Any commercial application

The framework is your **tool** - you own your work.

### Q: What if my legal department fears GPL?

**A: Common misconception.** GPL v3 only applies to the framework code itself, not to projects using it.
